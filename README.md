# KI-News-Feed

Dieser Ordner enthält den Tagesfeed für die Android-App.

- Repository: `matzee007-web/ki-news-feed`
- öffentliche Feed-Adresse: `https://raw.githubusercontent.com/matzee007-web/ki-news-feed/main/latest.json`
- Standardbranch: `main`
- Sichtbarkeit: öffentlich, damit die App ohne Anmeldung darauf zugreifen kann

Die Datei `latest.json` wird bei einer neuen Tagesausgabe vollständig ersetzt. Das Schema ist im App-Projekt unter `KI-News-Android/README.md` dokumentiert.
